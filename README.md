## 🇩🇴 ESPAÑOL

### 📌 Descripción
Análisis completo de ventas de la empresa de bebidas **Kola Real**, abarcando las 32 provincias de República Dominicana. El proyecto cubre el ciclo completo del dato: desde la limpieza en SQL hasta la visualización ejecutiva en Power BI.

### ❓ Problema
Los datos de ventas presentaban inconsistencias críticas incluyendo:
- Errores de codificación en nombres de provincias
- Espacios en blanco en campos clave
- Falta de estandarización en canales de venta
- Datos duplicados que afectaban el análisis

### ✅ Solución
1. **Limpieza y normalización** en SQL (PostgreSQL)
2. **Corrección de errores** en las 32 provincias
3. **Modelado de datos** en Power BI
4. **Creación de KPIs** utilizando lenguaje DAX
5. **Dashboard interactivo** con segmentación por canal y sabor

### 📊 Resultados
- Ganancia total identificada: **RD$208,000**
- El sabor **Cola** representa el **60%** de las ganancias totales
- **Santo Domingo** lidera en volumen de ventas
- El **canal colmado** es el principal canal de distribución
- Análisis geográfico completo de las 32 provincias

### 🛠️ Herramientas Utilizadas
| Herramienta | Uso |
|---|---|
| PostgreSQL | Base de datos y limpieza |
| SQL | Normalización y consultas |
| Power BI | Visualización y dashboard |
| DAX | Medidas y KPIs dinámicos |
| Excel | Análisis complementario |

## 🇺🇸 ENGLISH

### 📌 Description
Complete sales analysis for **Kola Real** beverage company, covering all 32 provinces of the Dominican Republic. The project covers the full data pipeline: from SQL cleaning to executive visualization in Power BI.

### ❓ Problem
The sales dataset contained critical inconsistencies including:
- Encoding errors in province names
- Whitespace in key fields
- Lack of standardization in sales channels
- Duplicate records affecting analysis accuracy

### ✅ Solution
1. **Data cleaning and normalization** using SQL (PostgreSQL)
2. **Error correction** across 32 provinces
3. **Data modeling** in Power BI
4. **KPI creation** using DAX language
5. **Interactive dashboard** with filtering by channel and flavor

### 📊 Results
- Total revenue identified: **RD$208,000**
- **Cola** flavor drives **60%** of total revenue
- **Santo Domingo** leads in sales volume
- **Colmado channel** is the primary distribution channel
- Complete geographic analysis across 32 provinces

### 🛠️ Tools Used
| Tool | Purpose |
|---|---|
| PostgreSQL | Database and cleaning |
| SQL | Normalization and queries |
| Power BI | Visualization and dashboard |
| DAX | Dynamic measures and KPIs |
| Excel | Complementary analysis |
